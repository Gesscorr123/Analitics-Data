# Geospatial Distance Analysis

## Overview
This project uses Python to calculate distances between geographic coordinates
in order to identify nearby locations and evaluate distribution feasibility.

Due to confidentiality policies, all data has been anonymized and replaced with
synthetic data.

## Key Features
- Distance calculation between coordinates
- Proximity filtering within a 500-meter radius
- Classification of locations by entity
- Export of results to Excel
